与螺柱的强度无关。一切都与保护电线有关。

在这种情况下，必须采取的保护措施是因为电线太靠近螺柱的边缘，容易受到电钻，螺钉和钉子的伤害。

如果电线位于常规2x4的中心（通常无需将钉子或螺钉拧得很深），通常不需要。
